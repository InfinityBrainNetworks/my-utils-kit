# my-utils-kit

A personal collection of day-to-day utility tools.

## Tools

| Package | Description |
|---------|-------------|
| [doc-maker](./packages/doc-maker) | Document creation and management |
| [file-arranger](./packages/file-arranger) | File organization and sorting |
| [shared](./packages/shared) | Shared utilities used across tools |

## Getting Started

```bash
git clone https://github.com/yourusername/my-utils-kit.git
cd my-utils-kit
npm install
```

## Structure

```
my-utils-kit/
├── packages/
│   ├── doc-maker/       # Document maker tool
│   ├── file-arranger/   # File arranger tool
│   └── shared/          # Shared helpers and utilities
└── package.json         # Workspace root
```
